# rocketlab_genai

Projeto que fornece um agente Text-to-SQL para análise de dados de e-commerce. O repositório agora contém:

- Backend (FastAPI) responsável por gerar SQL via LLM (Google Gemini) e executar queries em um banco SQLite.
- Frontend (React + Vite) com UI estilo moderno (branco/preto) para fazer perguntas e visualizar resultados.
